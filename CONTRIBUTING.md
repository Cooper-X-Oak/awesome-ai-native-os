# Contributing

Contributions should improve the atlas as a decision tool, not only add links.

## Good Contributions

Useful contributions usually do one of these:

```text
add a missing AI-native production system
improve a scene comparison
complete a case card
add evidence for a candidate
downgrade a weak project with a clear reason
clarify a reusable pattern
```

## Add A Project

When adding a project, include:

```text
name
official URL
business scene
core business object
production artifact
operating space
AI operation model
human control model
state and memory
delivery boundary
evidence links
grade suggestion
```

## Writing Style

Be concrete.

Prefer:

```text
This system turns a lead/account table into an AI-operable GTM workspace.
```

Avoid:

```text
This is an amazing AI tool for productivity.
```

## Evidence Discipline

Separate evidence from interpretation:

```text
Evidence: the project describes itself as a node-based AI studio for images, video, audio, and text.
Interpretation: this makes it a node-canvas-first multimodal production OS candidate.
```
