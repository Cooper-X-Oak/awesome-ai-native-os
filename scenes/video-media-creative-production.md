## English

### Business Core Abstraction

Turn scripts, prompts, assets, characters, voices, creative directions, timelines, and production stages into publishable media.

### What To Compare

| Axis | Questions / Difference |
| --- | --- |
| pipeline-first | OpenMontage splits media production into stages, tools, agent skills, and intermediate artifacts. |
| node-canvas-first | OpenFlow uses nodes and edges to organize multimodal generation. |
| timeline/transcript-first | Descript uses transcript and timeline as the editing surface. |
| creative-workspace-first | Krea and Runway center iteration around direction, variants, generation, and enhancement. |
| single-purpose workflow | OpusClip, HeyGen, and Synthesia optimize narrow media workflows. |

### 这一章要比较什么

| 维度 | 问题 / 差异 |
| --- | --- |
| pipeline-first | OpenMontage 把媒体生产拆成阶段、工具、agent skills 和中间产物。 |
| node-canvas-first | OpenFlow 用节点和连线组织多模态生成。 |
| timeline/transcript-first | Descript 用 transcript 和 timeline 作为编辑表面。 |
| creative-workspace-first | Krea 和 Runway 围绕创意方向、变体、生成和增强迭代。 |
| single-purpose workflow | OpusClip、HeyGen、Synthesia 面向更窄的媒体业务流程。 |
