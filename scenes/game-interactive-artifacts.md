# Game / Interactive Artifact Production

## Business Core Abstraction

Turn prompts or creative descriptions into playable games, interactive demos, HTML5 games, Pygame projects, or retro-style game artifacts.

## What To Compare

| Axis | Questions |
| --- | --- |
| Production object | Browser game, HTML5 game, Pygame game, interactive demo, retro game? |
| Operating space | Prompt pipeline, live preview, game editor, multi-agent pipeline? |
| Human control | Can the user edit mechanics, assets, state, or code? |
| Delivery boundary | Playable artifact, source package, hosted game? |
| Risk | Is it a demo, or a repeatable production workspace? |
