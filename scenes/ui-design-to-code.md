# UI Design / Design-to-Code / Front-End Delivery

## Business Core Abstraction

Turn UI ideas, designs, components, wireframes, or existing React interfaces into editable and deliverable front-end artifacts.

## What To Compare

| Axis | Questions |
| --- | --- |
| Design object | Is the core object a screen, component, React app, wireframe, or sitemap? |
| Interaction model | Is it visual editing, inspector editing, design import, or prompt-preview? |
| Code relationship | Does it generate code, modify existing app code, or export from design? |
| Human control | Can humans select, style, compose, inspect, and export? |
| Delivery boundary | Is the output a component, page, React code, or website plan? |
