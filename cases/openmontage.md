# OpenMontage

## One-Line Positioning

Pipeline-first AI-native video production OS.

## Business Scene

Video / media / creative production.

## Core Business Object

Staged video project: script, assets, production stages, tools, intermediate outputs, and final media package.

## Production Artifact

Publishable video or video production package.

## Operating Space

Production pipeline / board.

## AI Operation Model

Multi-agent planning, routing, composition, critique, and tool execution.

## Human Control Model

Human supervises stages, assets, task boundaries, and final output.

## State And Memory

Pipeline state, assets, scripts, tool calls, stage outputs, and execution traces.

## Delivery Boundary

Finished video production output.

## Reusable Pattern

Turn a creative production process into staged artifacts and repeatable AI-operated production steps.
