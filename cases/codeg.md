# CodeG

## One-Line Positioning

Multi-agent coding session OS.

## Business Scene

Software engineering / code delivery.

## Core Business Object

Code sessions and repository changes.

## Production Artifact

Code changes and project updates.

## Operating Space

Collaborative multi-agent coding workspace.

## AI Operation Model

Aggregate, route, supervise, and execute multiple coding agents or sessions.

## Human Control Model

Human manages sessions, inspects outputs, and supervises agent work.

## State And Memory

Session state, repo state, agent outputs, and task traces.

## Delivery Boundary

Code changes / coding deliverable.

## Reusable Pattern

Treat AI coding agents as parallel sessions that need a workspace, not only a chat box.
